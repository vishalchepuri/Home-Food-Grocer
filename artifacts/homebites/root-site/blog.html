Product story

Why HomeBites exists

HomeBites started as a food and grocery app inspired by how people actually browse: location first, cuisine next, then restaurants, menus, offers, cart, and checkout. The app also has an admin portal so catalog data can be managed without editing code.

  • Food app lives at /food-app/.
  • Restaurants and menus are served from Firebase through Cloud Run.
  • Reads are reduced with server caching and click-to-load sections.
  • The project deploys from GitHub Actions to Firebase Hosting and Cloud Run.
